# Consistent_CoT_NLP

### Drafting Extraction Rules
We can use the following framework as a starting point for our guidelines (and ask coming meeting whether it makes sense):
- **Explicit Commitments:** These must be direct quotes or near-direct paraphrases of factual claims made by the model. A rule of thumb can be here that if you cannot point to the specific words in the trace that make the claim, it is not explicit.
- **Implicit Commitments:** These are facts that must logically be true for the model's explicit statement to make sense, but are not directly stated. 
- **Logical Principles:** These are the overarching rules or principles the model applies to connect the excplicit and implicit commitments with final answers or intermediate steps.
